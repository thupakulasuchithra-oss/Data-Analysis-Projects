# Student Performance Analysis

This project analyzes student academic performance using Excel and Python.

## Objective
To analyze student marks and identify performance trends.

## Tools Used
- Excel
- Python (Pandas, Matplotlib)

## Features
- Calculated total and average marks
- Identified top and weak performers
- Visualized data using charts

## Files Included
- student_analysis.py
- Excel dataset
- Project report (PDF)
